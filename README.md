### 🔍 분석 방법
- 알고리즘 추천 시스템이 사회적 분열과 공감 형성에 미치는 영향을 데이터 기반으로 분석

- **부정적 사례 분석**: 정치적 성향에 따른 콘텐츠 편향과 필터버블 현상을 중심으로 연구
  - 정치 성향별 선호 언론사 분류 및 시각화
  - 이태원 참사 관련 언론 보도의 정치 성향별 프레임 차이 재해석
  - 디지털 뉴스 리포트를 활용해 국가별 유튜브 뉴스 이용률 도넛그래프 시각화
  - 논문 「유튜브 추천 알고리즘으로 인한 필터버블 현상 연구」 참고
  - 극성 유튜브 채널의 영상 제목 워드클라우드 분석으로 프레임 차이 확인
  
- **긍정적 사례 분석**: OTT 추천 알고리즘이 사회적 의제 형성과 제도 변화에 미친 영향 분석
  - OTT 이용률 추이 및 더 글로리 시청시간 변화 시각화
  - ‘학교폭력’ 키워드의 검색량, 보도량, 법안 발의 건수를 시계열로 비교
  - 법안 발의 및 본회의 통과 의안 정리

### 🧠 결론 및 미래전망
- 💔 분열 측면
  - 알고리즘은 사용자의 성향에 맞는 정보만 반복 노출해 필터버블과 확증편향을 강화하고, 잘못된 정보 확산(인포데믹)으로 사회적 극단화를 유발
- 🤝 공감 측면
  - OTT의 추천 시스템은 콘텐츠 흥행을 이끌며 저널리즘적 공론장 역할을 수행
- ⚙️ 종합적 시사점
  - 알고리즘은 분열과 공감의 경계선 위에 있음
    - **정부**는 EU의 **DSA(디지털서비스법)**처럼 플랫폼의 책임과 이용자 보호를 강화할 법적 장치 마련 필요
    - **콘텐츠 제작자**는 사회적 영향력을 고려해윤리적이고 책임 있는 제작 태도를 가져야 함
    - **개인**은 중요한 것은 "깨어있는 선택"이며,무비판적 소비가 아닌 의식적 콘텐츠 이용 태도가 요구
  - 미래의 미디어 환경은 알고리즘을 중심으로 재편될 것이며 정부의 제도적 관리, 제작자의 윤리적 책임, 이용자의 깨어있는 선택이 함께 어우러질 때 건강한 디지털 생태계가 가능할 것임
